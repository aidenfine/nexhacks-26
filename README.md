# TinyToken

**TinyToken** is a Chrome extension designed to enhance your AI chat experience on platforms like **ChatGPT**, **Claude**. It auto sends messages, and tracks your token savings to make your workflow faster and more efficient. TinyToken will compress your input without sacrificing the output given by the LLM.

## Features
- **Aggressiveness Slider**  
  Adjust how aggressive the API compresses your input TinyToken interacts with the chat. Set between **10% and 100%**.

- **Auto Send Messages**  
  Send your messages automatically, keeping things quick and easy.

- **Token Tracking**  
  Monitor the tokens saved as TinyToken optimizes your prompt usage, helping you stay efficient.

- **Supported Platforms**  
  Works with **ChatGPT**, **Claude**

## Usage

1. Open your AI chat interface (ChatGPT, Claude).  
2. Enter your prompt into TinyToken.

4. TinyToken will fill send prompt and track your saved tokens.
